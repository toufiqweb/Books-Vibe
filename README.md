#  Book Vibe

##Book Vibe – A Modern Book Management App 

## 🧠 Overview
Book Vibe is a modern web application built with React that allows users to explore books, view detailed information, and manage their personal reading lists.

## 🎯 Project Purpose
The goal of this project is to practice building a real-world React application with modern tools like Context API, routing, and local storage while creating a useful book management system for users

##  Key Features

### 1. **Book Exploration**
- Browse books loaded from an external API  
- View book covers, titles, and short descriptions  
- Clean and responsive UI for smooth browsing  

### 2. **Book Details & Navigation**
- View detailed information about each book  
- Seamless page navigation using React Router  
- Dynamic routing for individual book pages  
- User-friendly layout for better readability  

### 3. **Read List & Wish List Management**
- Add books to Read List  
- Add books to Wish List  
- Prevent duplicate entries  
- Persistent storage using Local Storage  
- Sync data across sessions automatically  



## 🚀 Run Locally

**Clone the project**
```bash
https://github.com/toufiqweb/Books-Vibe.git
```

**Go to the project directory**
   
```bash
cd Books-Vibe
```

**Install dependencies**
   
```bash
npm install
```
**Start the development server**
   
```bash
npm run dev
```
